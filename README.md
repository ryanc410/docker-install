## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

-Ubuntu Server

### Installation

1. Clone the repo
   git clone https://github.com/ryanc410/docker-install.git
2. Change to docker-install directory
  cd docker-install
3. Set as executable
  chmod +x docker-install.sh
4. Run it as root
  sudo ./docker-install
